Configurar busca ampliada (opcional)

A busca ampliada usa a API pública da Adzuna, que cobre vagas de qualquer empresa no Brasil, não só as monitoradas aqui. Precisa de uma chave gratuita:

  1. Acesse developer.adzuna.com e crie uma conta grátis
  2. Copie o App ID e a App Key gerados
  3. Cole abaixo e toque em "Salvar"

As chaves ficam salvas só neste navegador (não vão para o GitHub). Como o site é estático, elas ficam visíveis a quem inspecionar as requisições da página — não é um risco alto para uso pessoal, mas evite reusar chaves sensíveis aqui.
